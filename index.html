One quick setup step

Pocket Coach needs an Anthropic API key to power the AI. It's free to create — you only pay tiny amounts as you use it (a few cents per conversation).

1. Go to console.anthropic.com
2. Create a free account
3. Go to "API Keys" → "Create Key"
4. Copy and paste it below

Your key is saved only in your browser — it never leaves your device and is never stored on any server.
